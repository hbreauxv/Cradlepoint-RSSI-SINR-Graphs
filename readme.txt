Application Name
================
rssi_sinr_graphs

Application Version
===================
1.0


NCOS Devices Supported
======================
ALL


External Requirements
=====================
None


Application Purpose
===================
This application adds RSSI and SINR graphs into the router again.  They can be accessed by going to port 9001 on the router. 

Expected Output
===============
Port 9001 will display RSSI and SINR graphs
